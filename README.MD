# Firehorn II - Spine

<p>
  <img src="images/ert/fh_patch.png" alt="Firehorn Patch" width="150" align="right">
  This repository contains the layout and schematics for the Spine on Firehorn.
  The goal of the spine on the rocket is to connect all the main components
  together like the Flight Computer, Telemetry Board and the Propulsion Computer.
  It also interface with the external power and the batteries onboard the rocket.
</p>
